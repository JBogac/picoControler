# picoControler

> Komputer zbudowany z układów scalonych - projekt na elektronikę cyfrową

## Co to jest?

8-bitowy programowalny kontroler zbudowany z:
- Zegar: NE555
- Licznik: 74HC161 (0-15)
- RAM: HM6264 (16×8 bit)
- Moduły wykonawcze (LED, wyświetlacze, buzzer, w planach)
