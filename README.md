## Co to jest?

- Nazwa akademicka: Procesor w architekturze Harvardzkiej o architekturze RISC
- Nazwa praktyczna: 8-bitowy Mikrokontroler (Microcontroller Unit - MCU)
- Nazwa ze względu na technologię: Procesor TTL / Discrete Logic CPU
- Czyli po ludzku: coś co liczy, wykonuje operacje i daje się programować
- Używając 2x UCY74181 łączonych kaskadowo – 8-bitowa jednostka arytmetyczno logiczna
- Pamięć zbudowana z 2 kości EEPROM, łącznie 2048 x 16 bit
